# School-Project
Ho creato questo codice C++ per un compito scolastico assegnato dal professore di informatica.

Il programma permette di noleggiare E‑BIKE inserendo le proprie coordinate X e Y.
In base a queste informazioni, il software:

ordina e mostra a schermo le E‑BIKE più vicine

indica per ciascuna la distanza e l’autonomia disponibile

Una volta selezionata la E‑BIKE da noleggiare, il programma richiede di inserire i chilometri che si desidera percorrere.
Se l’autonomia è sufficiente:

viene calcolato e mostrato il tempo necessario

viene mostrato il costo totale del noleggio

Se invece l’autonomia non è sufficiente, il programma stampa un messaggio di errore.
